# Veo 3 이미지 입력 가이드

작성일: 2026-05-24
목표: PSI 스타일을 정확히 반영한 10초 영상 생성

## 결론
- 필수는 아님
- 하지만 권장: 예
- 이유: 이미지를 넣으면 결과가 PSI 화면 느낌에 훨씬 가깝게 나옴

## 권장 이미지 4장
1. REF_01_dashboard_hero
- 대시보드 전체가 보이는 대표 화면

2. REF_02_comparison_chart
- 비교/추세 차트가 잘 보이는 화면

3. REF_03_execution_card
- 실행 상태(미착수/진행중/완료) 카드가 보이는 화면

4. REF_04_logo_or_intro
- 로고 또는 소개 엔드카드 화면

## 이미지 준비 규칙
- 비율: 16:9 권장
- 해상도: 1920x1080 이상 권장
- 포맷: PNG 또는 JPG
- 화면 내 불필요한 팝업, 브라우저 UI, 민감정보는 제거
- 텍스트가 너무 작게 보이는 캡처는 피함

## 업로드 순서
1. Veo 3 프롬프트 입력창 열기
2. 생성한 프롬프트 파일 전체 붙여넣기
3. 이미지 첨부 버튼으로 4장 업로드
4. 파일명 또는 첨부 순서를 아래처럼 맞춤
- 1번: REF_01_dashboard_hero
- 2번: REF_02_comparison_chart
- 3번: REF_03_execution_card
- 4번: REF_04_logo_or_intro
5. 생성 실행

## 실패 줄이는 팁
- 1회차가 너무 추상적이면: 대시보드 캡처를 한 장 더 추가
- 텍스트 가독성이 낮으면: "Korean text legibility is top priority" 문구를 프롬프트 맨 아래 한 번 더 추가
- 색감이 다르면: 배경/텍스트/강조 색상 헥스코드를 다시 넣고 재생성

## 빠른 체크

- 마지막 1.5초 로고가 충분히 보이는지
- 자막 4구간이 잘 읽히는지
