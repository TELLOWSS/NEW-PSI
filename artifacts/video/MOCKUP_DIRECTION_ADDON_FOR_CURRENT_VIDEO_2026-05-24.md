# 목업 방향 전환 애드온 (현재 진행 영상용)

작성일: 2026-05-24
적용 대상: 현재 10초 PSI 소개 영상

## 1) 의견 요약
- 방향 전환은 매우 타당함
- 이유: 현재 목업은 제품 신뢰감, 정보 구조, 완성도를 한 프레임에서 보여줌
- 특히 임원/운영 리더 대상에서는 다크 시네마틱보다 밝은 제품형 프레임이 설득력이 높음

## 2) 무엇이 좋아졌는가
1. 제품 실체감 강화
- 데스크톱 대시보드 + 모바일 화면을 함께 보여줘 실제 도입 이미지를 빠르게 전달

2. 프리미엄 신뢰감 강화
- 밝은 배경, 정돈된 카드, 일관된 타이포가 엔터프라이즈 품질감을 만듦

3. 메시지 속도 개선
- 복잡한 설명 없이도 "통합-비교-실행" 흐름이 시각적으로 이해됨

## 3) 현재 10초 영상에 더할 수정 포인트
1. Scene 1
- 기존: 추상 데이터 중심
- 변경: 목업 보드 스타일(타이틀 + 모듈 카드)로 오프닝

2. Scene 2
- 기존: 차트 중심 단일 프레임
- 변경: 좌측 PC/우측 모바일 동시 배치

3. Scene 3
- 기존: 상태카드 추상 표현
- 변경: 목업형 실행 카드(미착수 -> 진행중 -> 완료) 명확 표시

4. Scene 4
- 기존: 다크 엔드카드
- 변경: 밝은 엔드카드 + PSI 로고 중심
