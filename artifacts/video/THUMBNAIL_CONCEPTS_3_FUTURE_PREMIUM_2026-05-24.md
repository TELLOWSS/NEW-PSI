# 썸네일 3종 시안 (Future Premium)

작성일: 2026-05-24
목표: 영상 클릭률과 브랜드 고급감 동시 확보

## 공통 가이드
- 캔버스: 1280x720
- 타이포: Pretendard ExtraBold + Sora SemiBold
- 색상: 딥 네이비 배경 + 실버 블루 포인트
- 금지: 과다 텍스트, 과한 효과, 3개 이상 강조색

## 시안 A: KPI 임팩트형
- 배경: dashboard KPI 카드 클로즈업
- 헤드라인: 더 빠른 결정
- 서브라인: 더 큰 성과
- 레이아웃: 좌측 텍스트 40%, 우측 KPI 60%
- 포인트 요소: KPI 카드 1개만 글로우 처리
- 사용 권장: 성과/실무 중심 메시지

## 시안 B: 비교 인사이트형
- 배경: performance-analysis 차트(레이더 또는 추세)
- 헤드라인: 팀 비교 인사이트
- 서브라인: 한 화면에서 즉시 판단
- 레이아웃: 중앙 차트 + 하단 짧은 카피
- 포인트 요소: 비교 축 1개 라인 콜아웃
- 사용 권장: 분석 기능 강조 메시지

## 시안 C: 실행 전환형
- 배경: predictive-analysis + intervention-coaching 분할
- 헤드라인: 보고, 비교, 실행
- 서브라인: PSI 운영 표준
- 레이아웃: 상단 카피, 하단 분할 화면
- 포인트 요소: 상태 배지(진행중/완료) 1개 강조
- 사용 권장: 의사결정-실행 연결 메시지

## 카피 A/B 테스트 문구
1. 더 빠른 결정, 더 큰 성과
2. 지표를 실행으로 바꾸다
3. 차세대 운영 표준 PSI

## 최종 선택 기준
- 모바일 미리보기에서 첫 1초에 문구가 읽히는가
- 배경 UI가 브랜드 신뢰를 주는가
- 카피가 10자 내외로 즉시 이해되는가
