# 10초 프리미엄 인트로 제작 체크리스트

작성일: 2026-05-24
목표: 초보자도 30분 내 완성 가능한 10초 버전
톤: 미래지향 프리미엄

## 1) 지금 상태에서 바로 쓸 페이지
- 1컷(통합): dashboard
- 2컷(비교): performance-analysis
- 3컷(실행): predictive-analysis
- 4컷(클로징): introduction 또는 로고 카드

## 2) 타임라인 고정값 (10.0초)
1. 00:00.0-00:03.5
- 페이지: dashboard
- 장면: KPI + 차트 동시 노출되는 히어로 프레임
- 모션: 103% -> 100% 느린 줌아웃
- 자막: 흩어진 지표를 하나의 흐름으로

2. 00:03.5-00:06.5
- 페이지: performance-analysis
- 장면: 추세 차트 또는 레이더 차트
- 모션: 좌->우 패닝(아주 느리게)
- 자막: 비교가 판단을 빠르게 만든다

3. 00:06.5-00:08.5
- 페이지: predictive-analysis
- 장면: 실행 계획 상태 카드(미착수/진행중/완료)
- 모션: 카드 중앙 고정 + 미세 줌인(100% -> 102%)
- 자막: 보고, 바로 실행한다

4. 00:08.5-00:10.0
- 페이지: introduction 또는 엔드카드
- 장면: 로고 + 제품명
- 모션: 라이트 스윕 1회
- 자막: PSI 차세대 운영 표준

## 3) 전환/효과 최소 규칙
- 클립 사이 전환: Motion Blur 0.25초만 사용
- 텍스트 애니메이션: Fade In Up 0.30초
- 금지: 글리치, 빠른 줌, 과한 흔들림

## 4) 색/사운드 최소 규칙
- 색보정: 대비 +10, 채도 -5, 선명도 +8
- BGM: 작게(-18dB 전후), 나레이션 없으면 -16dB
- 클릭/스윕 SFX는 최대 2~3개만 사용

## 5) 초보용 완료 기준
- 10초 정확히 맞춤
- 자막 4개 모두 읽힘
- 마지막 1.5초 로고/CTA 노출
- 무음 재생에서도 메시지 이해 가능
