# 10초 버전 분기안 (보고용 vs SNS용)

작성일: 2026-05-24
목표: 동일 소스(4컷)로 톤만 바꿔 2종 출력

## 1) 공통 소스/구조
- 컷 구조: dashboard -> performance-analysis -> predictive-analysis -> end card
- 길이: 10.0초
- 전환: Motion Blur 0.25초

## 2) 보고용 버전 (내부/임원 공유)
핵심 방향
- 신뢰/정확/안정 중심
- 과장 문구 최소화, 실행 문장 중심

자막 문구
1. 흩어진 운영 지표를
하나의 흐름으로 통합
2. 비교 기반 인사이트로
판단 시간을 단축
3. 보고에서 실행까지
한 화면으로 연결
4. PSI 운영 표준
지금 적용 시작

톤 설정
- 폰트 굵기: SemiBold
- 색상: 화이트 본문 + 실버블루 키워드 최소 강조
- BGM: -18dB, SFX 1~2개

## 3) SNS용 버전 (짧은 임팩트)
핵심 방향
- 속도/임팩트/기억점 중심
- 문장 짧게, 동사 강하게

자막 문구
1. 지표를
한 번에 본다
2. 비교하고
3초 안에 판단
3. 바로 실행
바로 변화
4. Faster Decisions
Higher Impact

톤 설정
- 폰트 굵기: Bold
- 색상: 화이트 + 키워드 블루 강조 빈도 증가
- BGM: -16dB, SFX 2~3개

## 4) 빠른 선택 기준
- 보고용: 회의/보고 자료, 설명 중심
- SNS용: 관심 유도, 클릭/재생 유지 중심
