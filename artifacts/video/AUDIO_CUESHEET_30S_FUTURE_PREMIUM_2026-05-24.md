# 30초 오디오 큐시트 (Future Premium)

작성일: 2026-05-24
기준: 30fps / 총 30.0초
목표: 나레이션 선명도 유지 + 프리미엄 긴장감 상승

## 1) 트랙 구성
- Track A (BGM Bed): 미니멀 신스 패드 + 저역 펄스
- Track B (Pulse): 킥 성격의 서브 펄스, 후반 강도 증가
- Track C (UI SFX): 카드 등장, 상태 전환, 스윕
- Track D (VO): 메인 내레이션
- Track E (Master FX): 엔드카드 리프트/테일

## 2) 타임라인 큐
1. 00:00.0-00:03.0

- VO: "흩어진 데이터를, 하나의 판단으로 연결합니다."
- 의도: 긴장감 있는 문제 제기

- VO: "지금, 차세대 운영 표준을 시작하세요."
- 의도: 명확한 CTA 착지

## 3) 믹싱 규칙
- VO 피크: -3dB 이하
- BGM 평균: -18~-14dB 구간
- SFX 평균: -22~-16dB 구간
- 덕킹: VO 구간 BGM 자동 -4dB
- 마스터 리미터: True Peak -1dB

## 4) 빠른 검수 체크
- 무음 재생 시 자막만으로 이해 가능한가
- 오디오만 들었을 때 장면 전환 타이밍이 들리는가
- 마지막 3초 CTA가 BGM에 묻히지 않는가
