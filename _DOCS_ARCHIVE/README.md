# _DOCS_ARCHIVE 아카이브

**목적**: 완료된 프로젝트, 과거 문서, 참고 자료 보관  
**대상**: 선택적 참고용 (필요할 때만)  
**관리**: PM (정기 정리)

---

## 📌 목적
- 현재 진행 중인 업무와 무관한 완료된 내용 분리
- 필요시 유사 사례 참고 가능성 유지
- 프로젝트 아카이빙 및 지식 보존

---

## 📂 구조

```
_DOCS_ARCHIVE/
├─ 2026-05/          (월별 구분)
│  ├─ 완료채크리스트_0521.md
│  ├─ 구현점검감사_0520.md
│  ├─ 실행계획_0516.md
│  └─ ... (80+ 파일)
│
├─ TOPICS/           (주제별)
│  ├─ 브랜드톤/
│  ├─ 성능최적화/
│  ├─ 보안검수/
│  └─ ...
│
└─ REFERENCE/        (참고 자료)
   ├─ 초기설계안/
   ├─ 벤치마크/
   └─ ...
```

---

## 📝 언제 아카이브하나?
- 프로젝트 완료 후
- 분기별 정리 (마지막주 금요일)
- 우선순위 변경 시 낙수 문서
