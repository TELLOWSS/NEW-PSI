## 3) OCR 분석(OCR)

### 3-1. 헤더
- 제목: `OCR 분석`
- 뒤로가기 라벨: `뒤로`

### 3-2. 히어로
- 메인 제목: `수기 기록지 OCR 분석`
- 보조 문구: `이미지 속 수기 기록지를 자동으로 인식하고 분석하여 현장 상태를 빠르게 파악합니다.`

### 3-3. 체크리스트
- `다국어 OCR 지원 (한국어, 영어, 중국어 등)`
- `손글씨 인식 최적화`
- `이미지 보정 및 왜곡 보정`
- `자동 위험 매핑`

### 3-4. 상태 문구
- 파일 선택 전: `분석할 이미지를 선택해 주세요.`
- 파일 선택 후: `분석 준비가 완료되었습니다.`
- 분석 중: `이미지를 분석하고 있습니다.`
- 완료 후: `분석이 완료되었습니다.`

### 3-5. 오류 문구
- 권한 거부: `카메라 또는 파일 접근 권한이 필요합니다.`
- 형식 오류: `지원하지 않는 파일 형식입니다.`
- OCR 실패: `문서 인식에 실패했습니다. 다시 시도해 주세요.`
- 수동 전환 안내: `필요하면 수동 입력으로 전환할 수 있습니다.`

### 3-6. CTA
- 기본: `분석 시작`
- 재시도: `다시 시도`
- 대체 액션: `수동 입력 전환`
