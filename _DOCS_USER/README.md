## 📱 화면별 빠른 가이드

### 1️⃣ 홈 대시보드
**이것부터 확인하세요:**
- "오늘 몇 명이 위험한가?" → KPI 카드 보기
- "가장 중요한 건?" → 위험 분포 (적색) 보기
- "최근 분석은?" → 최근 리포트 3개 보기

**할 수 있는 행동:**
- `시작하기` 버튼 → 새 분석 시작

### 2️⃣ 경보 알림
**이것부터 확인하세요:**
- "지금 누가 위험한가?" → 경보 리스트 보기
- "심각도 순서?" → 위에서부터 읽기 (빨강 > 노랑 > 초록)

**할 수 있는 행동:**
- `즉시 조치` 버튼 → 8번 개입 화면으로 이동

### 4️⃣ 위험인지 진단
**이것부터 확인하세요:**
- "내가 진행한 게 몇 개?" → 맨 위 진행도 카드 보기
- "다음 문항은?" → "다음" 버튼으로 진행

**할 수 있는 행동:**
- 텍스트 입력: 진단 문항에 답변 입력
- "다음" 버튼: 다음 문항으로 이동
- "완료" 버튼: 10개 모두 입력 후 제출

### 8️⃣ 개입 추천
**이것부터 확인하세요:**
- "즉시 뭘 해야 하나?" → 상단 빨간 카드 읽기
- "담당자는?" → 내 이름/팀 확인
- "기간은?" → "중기", "학습" 섹션 펼치기

**할 수 있는 행동:**
- `지정하기` 버튼: 내가 담당하겠다고 표시
- `완료` 버튼: 조치 완료했다고 표시

### 9️⃣ 수기 데이터 입력
**이것부터 확인하세요:**
- "어디까지 입력했나?" → 맨 위 진행도 보기 (3/10 완료?)
- "다음 문항은?" → "질문 N/10" 표시

**할 수 있는 행동:**
- 종이 기록지와 대조하며 입력
- 완료 후 검증 화면(10번)으로 자동 이동

## ❓ 자주 하는 질문 (FAQ)

### 기술 관련

**Q: 네트워크 끊기면 데이터 손실되나?**  
A: 아니요. 완료하기 전 입력 내용은 핸드폰에 자동 저장됩니다. 나중에 계속 가능.

**Q: 매번 로그인해야 하나?**  
A: 처음 한 번만. 그 다음부턴 자동 로그인.

**Q: 여러 기기에서 접속 가능한가?**  
A: 동시 접속 불가. 먼저 로그인한 기기가 끊깁니다.

### 사용법 관련

**Q: 진단을 꼭 다 해야 하나? 중간에 중단 안 되나?**  
A: 가능합니다. 중단했던 곳에서 "계속" 버튼으로 재개 가능. (그 세션만)

**Q: 잘못 입력했는데 수정할 수 있나?**  
A: 이전 화면으로 돌아가기 버튼으로 수정 가능.

**Q: 입력 완료했는데 돌아가고 싶으면?**  
A: 1번 홈 화면으로 돌아가세요. 입력 내용은 자동 저장.

### 데이터 관련

**Q: 내가 입력한 게 정말 저장되나?**  
A: 네. "저장 완료" 메시지 또는 체크마크(✓) 표시로 확인 가능.

**Q: 리포트는 누가 만드나?**  
A: 자동. 내가 입력 완료하면 시스템이 분석해서 리포트 생성.

**Q: 리포트는 몇 시간 후에 나와?**  
A: 대개 1-2시간. 복잡한 분석은 24시간 가능.

## 🛠️ 문제 해결 (Troubleshooting)

### "앱이 안 열려요"
1. 브라우저 새로고침 (F5 또는 새로고침 버튼)
2. 캐시 삭제: 브라우저 설정 > 캐시 및 쿠키 삭제
3. 다시 접속

### "로그인이 안 돼요"
1. 계정명 다시 확인 (띄어쓰기 없나?)
2. 비밀번호 다시 입력 (대소문자 구분)
3. 여전히 안 되면: 팀에 비밀번호 초기화 요청

### "데이터가 저장 안 된 것 같아요"
1. "저장 완료" 메시지 떴나 확인
2. 이전 화면 갔다가 다시 오기
3. 여전히 없으면: 운영팀에 보고

### "화면이 깨져 보여요"
1. 가로/세로 모드 맞나 확인? (세로 모드 권장)
2. 핸드폰 용량 확인? (저장 공간 부족하면 렉 발생)
3. 최신 버전? (앱 업데이트 확인)

### "버튼이 안 눌러져요"
1. 네트워크 연결 확인 (Wi-Fi 또는 모바일 데이터)
2. 로그인 상태 확인? (로그아웃 상태면 안 됨)
3. 잠시 기다렸다 다시 시도
