- [ ] 판정/코칭/채점요약이 다른 섹션을 침범함

## 긴급 재검증 (국적 3케이스 필수)

아래 3건은 이번 반영사항(모국어 제목/앞장 밀도 보강/6대 지표 병기/코칭 비축약/하단 한글화) 확인을 위한 필수 점검입니다.

### Case-KR (한국인)
- [ ] 앞장 최상단 제목 하단 모국어 표기(한국어) 정상
- [ ] 앞장 요약 영역이 과도 축약 없이 의미 단위로 읽힘
- [ ] 뒷장 6대 지표가 한국어 중심으로 가독성 유지
- [ ] 현장실행우선 코칭 문구가 줄임말 없이 완전 문장으로 표시
- [ ] 하단 영문 표기가 한국어로 변환되어 표시
- [ ] 텍스트/그리드 넘침 없음

### Case-VN (베트남인)
- [ ] 앞장 최상단 제목 하단 베트남어 표기 정상
- [ ] 앞장 요약(채점요약/코칭요약/판정)에서 문단이 과도 축약되지 않음
- [ ] 뒷장 6대 지표가 베트남어+한국어 병기로 표시되고 그리드 넘침 없음
- [ ] 현장실행우선 코칭 모국어 문장이 완전 문장으로 전달됨
- [ ] 하단 영문 안내가 한국어 안내로 표기
- [ ] 텍스트 줄바꿈/클램프 적용 시 의미 손상 없음

### Case-CN (중국인)
- [ ] 앞장 최상단 제목 하단 중국어 표기 정상
- [ ] 앞장 요약 영역 내용 밀도가 빈약하지 않음(핵심 근거 확인 가능)
- [ ] 뒷장 6대 지표가 중국어+한국어 병기로 표시되고 격자 넘침 없음
- [ ] 현장실행우선 코칭 모국어 문장이 줄임 없이 전달됨
- [ ] 하단 영문 표기가 한국어로 표기됨
- [ ] 페이지 경계/패널 경계 텍스트 침범 없음
