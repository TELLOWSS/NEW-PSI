## 9. 시스템 안정성 및 사용자 지원 정보 (v2.1)

최신 v2.1 업데이트는 대규모 산업 현장의 요구사항을 반영하여 설계되었습니다.

- **대규모 처리 성능**: Progressive Rendering 엔진 최적화를 통해 300명 이상의 근로자 데이터를 1~3초 이내에 안정적으로 렌더링합니다. ($BATCH\_SIZE = 5$ 적용)
- **API 안정성**: AI 호출 할당량(Quota)을 실시간 추적하여 429 에러(Rate Limit) 발생 시 자동 백오프(Backoff) 및 60초 냉각 모드를 실행합니다.
- **모바일 최적화**: 햄버거 메뉴와 터치 타겟(44x44px) 최적화를 통해 현장 어디서든 스마트폰으로 관리 업무가 가능합니다.

## 10. [부록] 용어 사전 및 필수 체크리스트

### 📑 핵심 용어 정의

- **개인 안전역량(Competency)**: 심리, 이해도, 무결성 등을 정량화한 분석 점수($P$)
- **안전 등급(Grade)**: 역량 점수와 신뢰도를 기반으로 부여된 시각적 상태(Green/Yellow/Red)
- **증빙 패키지**: 사고 발생 시 법적 방어권을 보장하기 위해 분석부터 승인까지의 전 과정을 묶은 데이터 체인

### ✅ 실무자 체크리스트

- [ ] 이미지 품질: 기록지의 글씨와 필압이 판독 가능할 정도로 선명한가?
- [ ] API 할당량(Quota) 확인: 대량 분석 전 시스템 설정에서 API 가용 상태를 확인했는가?
- [ ] 매칭 오류 점검: 신뢰도 미달 또는 근로자 미매칭 데이터가 대기열에 남아있는가?
- [ ] 조치 기록 확인: 초급 등급 근로자에 대한 재교육 결과가 증빙 패키지에 포함되었는가?
