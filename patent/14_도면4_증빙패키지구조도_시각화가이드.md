## 법규 대응 체크포인트

| 항목 | 증빙 위치 | 중대재해법 관련 조항 |
|------|-----------|---------------------|
| 위험성평가 실시 | 원본+OCR결과 | 제4조 안전보건관리체계 |
| 근로자 의견 수렴 | 서명·날짜 | 제4조 1항 4호 |
| 조치 이행 확인 | 조치이력+완료증빙 | 제4조 1항 2호 |
| 관리감독자 점검 | 승인이력 | 제4조 1항 1호 |
| 기록 보존 | 전체 패키지 | 산업안전보건법 시행령 |

## 검색 쿼리 예시

```sql
-- 특정 기간 내 초급 등급 근로자 조회
SELECT * FROM evidence_packages 
WHERE index.month = '2026-02' 
  AND index.safetyLevel = '초급'
ORDER BY createdAt DESC;

-- 미완료 조치 건수 집계
SELECT team, COUNT(*) 
FROM evidence_packages 
WHERE actions.completedAt IS NULL
GROUP BY team;
```
