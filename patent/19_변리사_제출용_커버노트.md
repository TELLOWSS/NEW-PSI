# 변리사 제출용 커버노트

## 0. 문서 관리 정보
- 발명 및 개발 총괄: 박성훈
- 작성일: 2026-04-09
- 시스템 적용 버전: PSI v2.2.0
- 상태: ✅ 변리사 전달용 정리본
- 최종본 잠금 태그: PSI-v2.2.0-2026-04-09-AMENDED

## 1. 전달 목적
본 커버노트는 PSI 특허 출원 패키지의 전문 보정 반영 사항을 변리사 검토 관점에서 빠르게 파악할 수 있도록 요약한 문서입니다.

## 2. 발명 개요 요약
- 발명명: 수기 위험성평가 기록의 OCR 기반 현장 위험신호 해석, 역할 적응형 안전 피드백 제공, 개인 안전역량 산정 및 감사가능 증빙 생성 시스템 및 그 방법
- 핵심 기술:
  1. 수기 기록 OCR 구조화
  2. 개인 안전역량 산정
  3. 의미 상태 생성
  4. 역할 적응형 결과 표현 변환
  5. 증빙 패키지 및 무결성 검증

## 3. 이번 전문 보정 핵심
### 3-1. 명세서 측면
- 역할 적응형 표현 정책 모듈 추가
- 역할별 전달 인터페이스 모듈 추가
- 의미 상태 변환 단계(S155/S158 계열 취지) 반영
- 메시지 정책 버전 및 표현 이력 저장 근거 반영

### 3-2. 청구범위 측면
- 방법 독립항에 아래 단계 추가
  - 의미 상태 생성
  - 사용자 역할별 결과 표현 변환
  - 결과 표현 또는 정책 버전 저장
- 시스템 독립항에 아래 모듈 추가
  - 역할 적응형 표현 정책 모듈
  - 역할별 전달 인터페이스 모듈
- 종속항에 아래 포인트 추가
  - 비징계성 코칭 표현 치환
  - 역할별 용어 사전/상태명 세트 적용
  - 메시지 정책 버전 저장

### 3-3. 도면 측면
- 도 1: 역할 적응형 표현 정책 모듈(160), 역할별 전달 인터페이스 모듈(195), 리포트 생성 모듈(196)
- 도 2: 의미 상태 변환, 역할 기반 결과 표현 생성 단계 추가
- 도 4: 표현 정책 이력 저장부(355) 추가

## 4. 변리사 우선 검토 요청 사항
1. 역할 적응형 표현 정책이 단순 UI 구성 변경이 아니라 기술적 처리 구조로 충분히 청구되는지 검토 요청
2. 의미 상태 생성 및 정책 버전 저장 요소가 방법항/시스템항에서 과도하게 기능적 표현으로 보이지 않는지 검토 요청
3. 보호구 표지 연동 포인트를 본 출원에 유지할지, 분할 후보로 분리할지 검토 요청
4. 청구항 5~6의 표현 정책 관련 종속항과 기존 표지 연동 포인트 간 우선순위 검토 요청
