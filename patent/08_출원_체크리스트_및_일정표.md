## 1. 제출 전 체크리스트
- [ ] 발명의 명칭 확정(국문/영문)
- [ ] 발명자/출원인 정보 확정
- [ ] 명세서 본문(기술분야/배경/과제/해결수단/효과) 검토
- [ ] 청구항 독립/종속 구조 검토
- [ ] 도면(블록도/흐름도/부호) 정합성 확인
- [ ] 국문초록/영문초록 일치성 확인
- [ ] 선행기술 조사 결과 반영
- [ ] 실증 데이터(초기 값이라도) 첨부
- [ ] 가중치 버전관리(w1~w6) 문구가 명세서·청구항·초록·진보성 문서에 일치 반영됨
- [ ] 승인권자 역할 기반 승인정책 분기 문구가 명세서·청구항·진보성 문서에 일치 반영됨
- [ ] 정정 전/후값 및 증빙 패키지 해시 무결성 문구가 명세서·청구항·스키마 문서에 일치 반영됨
- [ ] 비밀유지/NDA 및 공개자료 점검
- [ ] 변리사 최종 검토 완료

## 2. 권장 일정(예시, 4주)
### 1주차
- 발명자 인터뷰
- 선행기술 1차 조사
- 명세서 1차 초안 확정

### 2주차
- 청구항 고도화
- 도면 작성/보정
- 초록 및 요약자료 확정

### 3주차
- 실증자료/KPI 보강
- 내부 법무/경영 검토
- 변리사 피드백 반영

### 4주차
- 최종 서류 패키징
- 출원서 제출
- 후속 OA 대응 전략 수립

## 3. 출원 후 체크

- [ ] 해외출원(PCT) 여부 의사결정
- [ ] 상표/디자인권 연계 검토
