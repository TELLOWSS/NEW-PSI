## Mermaid 다이어그램 코드

```mermaid
flowchart TB
    subgraph INPUT["이미지 입력 모듈 (110)"]
        A1[모바일 촬영]
        A2[스캐너 입력]
    end
    
    subgraph OCR["OCR 모듈 (120)"]
        B1[필기문자 인식]
        B2[신뢰도 산출]
        B3[항목 구조화]
    end
    
    subgraph NORMAL["정규화 모듈 (130)"]
        C1[작업유형 매핑]
        C2[위험요인 분류]
    end
    
    subgraph PROFILE["개인 프로파일링 모듈 (140)"]
        D1[심리 지표]
        D2[업무 이해도]
        D3[위험성평가 이해도]
        D4[숙련도]
    end
    
    subgraph GRADE["안전등급 산정 모듈 (150)"]
        E1[가중치 적용]
        E2[점수 계산]
        E3[등급 판정<br/>고급/중급/초급]
    end
    
    subgraph EXPRESS["역할 적응형 표현 정책 모듈 (160)"]
        F1[의미 상태 변환]
        F2[메시지 사전 적용]
        F3[역할별 출력 생성]
    end
    
    subgraph DASH["관리자 대시보드 모듈 (170)"]
        G1[위험신호]
        G2[개선 우선순위]
        G3[추세 분석]
    end
    
    subgraph EVIDENCE["증빙 저장 모듈 (180)"]
        H1[원본 이미지]
        H2[정정 이력]
        H3[조치 이력]
        H4[무결성 검증]
    end
    
    subgraph BADGE["보호구 표지 연동 모듈 (190)"]
        I1[등급 표기 생성]
        I2[QR/NFC 갱신]
    end

    subgraph CHANNEL["역할별 전달 인터페이스 모듈 (195)"]
        J1[근로자용 보완 권고]
        J2[관리자용 검토 화면]
        J3[경영진용 예방 지표]
    end

    subgraph REPORT["리포트 생성 모듈 (196)"]
        K1[감사 리포트]
        K2[기간별 집계]
    end
    
    INPUT --> OCR
    OCR --> NORMAL
    NORMAL --> PROFILE
    PROFILE --> GRADE
    GRADE --> EXPRESS
    EXPRESS --> DASH
    EXPRESS --> CHANNEL
    GRADE --> BADGE
    GRADE --> EVIDENCE
    DASH --> EVIDENCE
    CHANNEL --> EVIDENCE
    EVIDENCE --> REPORT
    
    style INPUT fill:#e3f2fd

    style BADGE fill:#ede7f6

    style REPORT fill:#fff8e1
```

## 도면부호 범례
- **100**: 전체 시스템
- **110**: 이미지 입력 모듈
- **120**: OCR 모듈
- **130**: 정규화 모듈
- **140**: 개인 프로파일링 모듈
- **150**: 안전등급 산정 모듈
- **160**: 역할 적응형 표현 정책 모듈
- **170**: 관리자 대시보드 모듈
- **180**: 증빙 저장 모듈
- **190**: 보호구 표지 연동 모듈
- **195**: 역할별 전달 인터페이스 모듈
- **196**: 리포트 생성 모듈

## 색상 가이드 (출원 흑백본 대비용)
- 입력/출력 블록: 연한 청색
- 처리 블록: 연한 황색
- 저장 블록: 연한 녹색
- 연동 블록: 연한 자주색
