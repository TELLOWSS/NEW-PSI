## 12) 예상 질문과 답변

**Q1**: "왜 ±8?" / "이건 설정 기준은 뭔가요?"
A: 같은 공종에서 실무 전문가 2인의 판정이 보통 ±5~7점 차이납니다. 여기에 안전마진 1~2점을 더해 ±8로 설정했으며, 월간 신뢰성 검증을 통해 지속 조정 가능합니다.

**Q2**: "근거가 없다면 어떻게?"
A: 근거 없는 점수는 시스템에서 우선 반려됩니다. 요청 시 다시 근거를 기록하게 강제되며, 이를 통해 현장 평가의 성의도가 높아집니다.

**Q3**: "비용이 정말 절감되나?"
A: 전수 OCR 대신 텍스트 기반 백필을 우선하되, 구조화 데이터가 충분하면 완전 백필도 가능합니다. 결과적으로 4월 기준 23.53% 절감을 달성했습니다.
