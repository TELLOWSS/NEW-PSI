#### 6-2-2. 로딩 상태

**시나리오 2: 느린 네트워크 (3초+ 로딩)**

- [ ] **로딩 표시 (1초 이상)**
  - [ ] 스켈레톤 로더 표시
  - [ ] 텍스트: "로딩 중..."
  - [ ] 애니메이션 부드러운가?

- [ ] **타임아웃 처리 (> 10초)**
  - [ ] 취소 버튼 표시
  - [ ] 오류 메시지: "로드 시간 초과"
  - [ ] `재시도` 버튼

**테스트 방법**: 브라우저 DevTools → Network 탭 → "Slow 3G" 선택

#### 6-2-3. 오류 상황

**시나리오 3: 네트워크 오류 또는 서버 오류**

- [ ] **오류 메시지 표시**
  ```
  ⚠️ 데이터 로드에 실패했습니다.
  
  [재시도] [도움말]
  ```

- [ ] **경고 토스트 (우측 하단)**
  - [ ] 3초 후 자동 사라짐
  - [ ] 수동 닫기 버튼 (X)

- [ ] **로그 기록** (Console)
  - [ ] 오류 메시지 상세 로그
  - [ ] 타임스탐프 포함

**테스트 방법**: DevTools → Network 탭 → 요청 우클릭 → "Block request URL"

#### 6-2-4. 권한/인증 오류
