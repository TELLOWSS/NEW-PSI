# PSI 소개영상 제작 패키지 (미래지향 프리미엄 톤)

작성일: 2026-05-24
목표 길이: 30초 메인 + 15초 숏폼
톤: Future-forward, Premium, Clean Tech

## 1) 핵심 메시지
- 데이터는 많아도, 결정은 더 빨라져야 한다.
- PSI는 흩어진 운영 신호를 한 흐름으로 통합해 실행 가능한 판단으로 바꾼다.
- 결과적으로 팀은 더 짧은 시간에 더 정확한 결정을 내린다.

## 2) 30초 영상 구성 (타임라인)
1. 0.0-3.0s: 문제 제기
- 화면: 어두운 공간 위에 분절된 데이터 레이어가 떠다니며 미세 노이즈.
- 자막: 정보는 넘치는데, 결정은 왜 늦을까

2. 3.0-8.0s: 통합의 시작
- 화면: 데이터 조각들이 하나의 대시보드 UI로 자기 정렬.
- 자막: 모든 지표를 하나의 흐름으로

3. 8.0-15.0s: 핵심 가치 3연타
- 화면: 실시간 모니터링 -> 팀 비교 -> 실행 포인트를 매치컷으로 연결.
- 자막: 실시간 모니터링 | 팀 비교 인사이트 | 즉시 실행 포인트

4. 15.0-23.0s: 전환 전후 대비
- 화면: 기존 복잡한 리포팅 플로우(좌)에서 간결한 PSI 화면(우)로 크로스.
- 자막: 복잡한 보고에서, 명확한 실행으로

5. 23.0-27.0s: 성과 집중
- 화면: KPI 카드 2~3개가 순차 확대, 수치 강조 애니메이션.
- 자막: Faster Decisions. Higher Impact.

6. 27.0-30.0s: 클로징/CTA
- 화면: 로고 + 제품명 + URL/문의 채널
- 자막: 지금, 차세대 운영 표준을 시작하세요

## 3) 내레이션 원고
### 30초 버전
흩어진 데이터를, 하나의 판단으로 연결합니다.
실시간으로 보고, 비교하고, 바로 실행합니다.
더 적은 시간으로, 더 정확한 결정을 만듭니다.
의사결정의 속도가 성과의 격차를 만듭니다.
지금, 차세대 운영 표준을 시작하세요.

### 15초 숏폼 버전
흩어진 데이터를 하나의 판단으로.
실시간으로 보고, 비교하고, 실행합니다.
더 빠른 결정, 더 큰 성과.
지금 PSI를 시작하세요.

## 4) 컷별 캡처 가이드 (실무용)
1. 메인 대시보드 히어로 화면: 4~5초
- KPI 카드 3개 이상이 동시에 보이는 프레임.
- 상단 내비/필터가 보이게 캡처해 신뢰감 확보.

2. 팀 비교 화면: 3~4초
- 비교 축이 명확한 영역(랭킹/증감/색상 대비) 중심으로 팬.

3. 실행 포인트 화면: 3~4초
- 사용자 행동이 일어나는 버튼/상태 변화 장면 포함.

4. 모바일 화면(있다면): 2~3초
- 짧게 삽입해 modern stack 인상 강화.

5. 로고 엔드카드: 2~3초
- 배경은 딥 네이비 그라디언트 + 아주 약한 라이트 스윕.

## 5) 비주얼 스타일 가이드
- 컬러
  - Background: #07121D
  - Surface: #0D1B2A
  - Text Primary: #EAF2FF
  - Accent Silver Blue: #9FC4FF
  - Accent Mint Highlight: #9EF0D0

- 타이포
  - 권장: Pretendard (KR) + Sora (EN 키워드)
  - 제목: SemiBold, 자간 +1~2%
  - 본문: Regular, 행간 130~145%

- 모션
  - 기본 전환: 10~14프레임 ease-out
  - 요소 등장: 20~40px 슬라이드 + 0.25~0.4s fade
  - 금지: 과한 글리치, 과한 줌, 잦은 흔들림

## 6) 사운드 디자인
- BGM: 미니멀 신스 + 저역 펄스(100~115 BPM)
- SFX
  - 카드 등장: 소프트 클릭
  - 수치 강조: 짧은 에어 스웰
  - 엔드카드: 저역이 살짝 올라오는 리프트
- 믹싱
  - 나레이션 우선, BGM은 -16~-20 LUFS 구간 권장

## 7) 편집 체크리스트
- 한 화면 한 메시지 원칙 유지
- 자막은 1~2줄, 읽기 시간 최소 1.2초 확보
- 숫자/KPI는 강조 색 한 가지로 통일
- 27초 이후 CTA는 단일 메시지만 노출
- 최종 검수: 무음 상태에서도 메시지 이해 가능해야 함
