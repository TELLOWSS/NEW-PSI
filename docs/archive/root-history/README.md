# 루트 누적 문서 아카이브

2026-07-17 상품화 검토 승인에 따라 루트에 누적된 과거 계획서, 완료 보고서, 회차별 인계서 126개를 이 폴더로 이동했습니다.

- 삭제하지 않고 원문을 보존합니다.
- 현재 실행·배포·운영에 직접 쓰는 문서는 저장소 루트 또는 `docs`에 유지합니다.
- 특허, 감사, 보안, 마이그레이션, `reports`, 기존 `archive` 자료는 이동 대상에서 제외했습니다.
- 과거 문서의 상대 링크는 작성 당시 구조를 기준으로 하므로, 현재 코드는 저장소 검색을 함께 사용해 확인합니다.

현재 기준 문서의 시작점은 저장소 루트의 [README.md](../../../README.md)입니다.
