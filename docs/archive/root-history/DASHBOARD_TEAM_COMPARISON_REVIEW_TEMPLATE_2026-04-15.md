## 2) 화면 문구 검토 체크리스트

### 핵심 문구
- [ ] 상단 안내 문구가 첫 액션(공종 → 팀선택)을 명확히 안내한다.
- [ ] 헤드라인이 `A팀 vs B팀 vs C팀` 형태로 비교 구조를 직관적으로 보여준다.
- [ ] 기준 배지가 `전체 국적 통합`으로 유지된다.
- [ ] 팀 미선택 상태에서 `국적 보기`는 비활성 + 안내 문구가 나온다.
- [ ] 선택 팀이 있을 때 `선택 1팀 국적 보기` 문구가 이해 가능하다.

## 3) 토글 위치/동선 체크리스트

- [ ] 고정 상단에서 `공종 선택/팀 선택/팀 비교 시작` 흐름이 끊기지 않는다.
