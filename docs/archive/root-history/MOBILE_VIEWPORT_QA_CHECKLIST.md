- [ ] 콘솔 에러 0건 확인

## 1) 공통 페이지 검수 항목 (모든 페이지)

- [ ] 주요 CTA 버튼이 화면 밖으로 넘치지 않음
- [ ] 입력창 포커스 시 키보드로 인해 입력영역이 가려지지 않음
- [ ] 표/카드가 가로 스크롤 또는 줄바꿈으로 읽기 가능 상태 유지
- [ ] 모달 열림/닫힘 시 배경 스크롤 잠금이 정상
- [ ] 텍스트 잘림(ellipsis 의도 외) 없음
- [ ] 터치 타겟(버튼/체크박스/셀렉트) 오터치 없이 동작

## 2) 페이지별 수동 검수

### Dashboard
- [ ] KPI 카드가 1열~2열 반응형으로 자연스럽게 재배치
- [ ] 차트 범례/축 텍스트 겹침 없음
- [ ] 상단 이동 버튼(페이지 전환) 터치 정상
- [ ] 팀 비교 상단 배지가 320/360 폭에서 1줄로 유지됨 (`공종` 라벨 우선)
- [ ] 비교 근거 카드 문구가 320/360 폭에서 2줄 내로 가독성 유지됨

### OcrAnalysis
- [ ] 상단 액션 버튼 그룹(재분석/불러오기/백업/삭제) 모바일에서 버튼 폭/정렬 정상
- [ ] 검색/팀장필터/AI갱신 버튼이 줄바꿈 상태에서 사용 가능
- [ ] 테이블 컬럼이 375 폭에서도 최소 정보 읽기 가능
- [ ] 신규 기록 분석 버튼이 화면 밖으로 넘치지 않음

### WorkerManagement
- [ ] 근로자 카드/리스트가 1열 기준으로 읽기 가능
- [ ] QR 관련 버튼/미리보기가 작은 화면에서 겹치지 않음

### PredictiveAnalysis
- [ ] 예측 카드/권고 텍스트가 화면폭에 맞춰 줄바꿈
- [ ] 그래프/인사이트 박스 겹침 없음

### PerformanceAnalysis
- [ ] 필터/정렬 컨트롤이 모바일에서 수직 정렬로 사용 가능
- [ ] 차트와 표 전환이 터치로 정상 동작

### SafetyChecks
- [ ] 점검 입력 폼(사유/상세) 모바일 입력 동선 정상
- [ ] 저장/취소 버튼 전체폭 또는 적절한 간격 유지

### SiteIssueManagement
- [ ] 이슈 목록 카드 클릭 영역 정상
- [ ] 상태 배지/버튼 줄바꿈 시 레이아웃 깨짐 없음

### Reports
- [ ] 필터(팀/등급/기간) 설정 영역이 모바일에서 사용 가능
- [ ] 미리보기 모드에서 좌우 네비게이션 버튼 터치 가능
- [ ] 단건/일괄 생성 버튼이 화면 밖으로 넘치지 않음

### IndividualReport
- [ ] 프로필/리포트 캡처 영역이 모바일에서 잘리지 않음
- [ ] 이미지 저장/PDF 발급 버튼 동작 정상

### Feedback
- [ ] 폼 입력/선택 UI가 키보드 오버레이에 가려지지 않음
- [ ] 제출 버튼 터치 정상

### Introduction
- [ ] 카드/텍스트 섹션 가독성 유지
- [ ] 내부 링크/버튼 터치 정상

### Settings
- [ ] API 키/현장정보/공종 입력 폼 모바일 배치 정상
- [ ] 가중치 입력(w1~w6) 1열 또는 2열로 읽기 가능
- [ ] 가중치 합계 상태 배지(권장/이탈) 표시 정상
- [ ] 가중치 이력 목록 + ‘이 버전 복원’ 버튼 터치 정상
